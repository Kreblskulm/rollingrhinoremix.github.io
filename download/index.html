Rolling Rhino Remix - Minimal ISO

The minimal disk image is for those who need absolute control over their system. If you wish to use another desktop environment other than Ubuntu-desktop, or want control over which packages are installed, than this disk image is for you. The minimal version is built upon Ubuntu Server and provides a more Arch Linux style of installation.
